# Linux Shell

A simple Linux shell developed in C as part of Linux System Programming.

## Features

- Execute Linux commands
- Built-in commands
  - cd
  - hist
  - curpid
  - ppid
  - quit
- Input Redirection (<)
- Output Redirection (>)
- Pipe (|)
- Multiple commands (;)
- Command History
- Makefile support

## Technologies Used

- C Programming
- Linux System Calls
- GCC
- Makefile

## Project Structure

```
main.c
builtin.c
history.c
execute.c
pipe.c
header.h
Makefile
```

## Build

```bash
make
```

## Run

```bash
./shell
```

## Clean

```bash
make clean
```
